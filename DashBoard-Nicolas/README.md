
# Dashboard CP1

# Como rodar
1. Instalar dependências:
pip install pandas numpy matplotlib seaborn streamlit scipy

2. Rodar o dashboard:
streamlit run dashboard.py

O dashboard contém as abas:
- Home: apresentação pessoal e objetivo profissional
- Formação & Experiência: cursos, certificações e experiências
- Skills: hard e soft skills
- Análise de Dados: análise completa do dataset de e-commerce
